## 🎉 Minimaldoro v1.0.0 — Stable Release

**Elegant desktop countdown timer and event tracker for Windows.**

### What's New in v1.0.0
- **Fixed** app not reopening after close — window now properly shows when relaunched from desktop or Start Menu
- **Updated** app icon to new logo across all platforms
- **Bumped** to stable 1.0.0

### Features (from v0.1.0)
- **Event Management** — Add, edit, and delete countdown events with custom colors and reminders
- **Desktop Widgets** — Pin events to your desktop as resizable, always-on-top windows
- **System Tray** — Runs quietly in the background with quick actions
- **Auto-start on Login** — Widgets appear automatically when you log in
- **Notifications** — Native Windows notifications before events arrive
- **Auto-updater** — Get new versions automatically via GitHub Releases
- **Privacy First** — All data stored locally in SQLite; no telemetry, no cloud sync

### Download
Download the installer from the [Releases page](https://github.com/psb-001/minimaldoro/releases).

### Known Limitations
- Windows-only (macOS/Linux coming later)
- No code signing — Windows SmartScreen may show warnings on first install
