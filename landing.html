How it works
Three steps to never
forget what matters
1
Add your events
Create countdowns for birthdays, deadlines, anniversaries — anything you want to track. Set colors and reminders.
2
Pin to desktop
Click the pin icon on any event to create a beautiful always-on-top widget right on your desktop.
3
Never miss out
Get native Windows notifications before events. Widgets persist across reboots — they're there when you need them.

FAQ
Questions? Answered.

Yes, 100% free and open source under the MIT License. No hidden costs, no premium tier, no data selling.

Currently Windows only. macOS and Linux support is on the roadmap. The tech stack (Electron + Svelte) is cross-platform, so future ports are straightforward.

Absolutely. All your data is stored locally in a SQLite database on your machine. Nothing is sent to any server — no telemetry, no analytics, no cloud sync.

Click the pin icon on any event to create a desktop widget. Widgets are always-on-top, resizable, and persist across app restarts. Right-click a widget to unpin it.

When creating or editing an event, set a reminder (2 hours, 3 hours, 6 hours, or 1 day before). Minimaldoro will send a native Windows notification at the right time.

Minimaldoro checks for updates automatically on launch. When an update is available, it downloads in the background and notifies you to restart from the tray menu.

Of course! The project is open source on GitHub. Fork it, create a feature branch, and open a Pull Request. Bug reports and feature requests are welcome via GitHub Issues.
